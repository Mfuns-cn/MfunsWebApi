## 更换背景



### 地址

```url
/module/action/skin.php
```

### 要求
- 用户已登录
- 非黑名单用户

### 方法
- POST


### 参数

| 参数      | 类型 | 值  | 备注     |
| --------- | ---- | --- | -------- |
| number | 整数 |     | 背景id|
|author_id|整数||要更换背景的用户（仅管理员可用，普通用户为自己的id）|

### 返回

| 参数 | 类型   | 值    | 备注 |
| ---- | ------ | ----- | ---- |
| code | 整数   | `0`(失败),`1`(成功)| 请求的状态 |
| msg  | 字符串 |       |   状态的具体说明 |

```json
{
    "code": 1,
    "msg": "设置成功"
}
```
