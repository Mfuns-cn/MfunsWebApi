## 领取签到奖励



### 地址

```url
/module/action/sign-treasure.php
```

### 要求
- 用户已登录
- 非黑名单用户

### 方法
- POST


### 参数

| 参数   | 类型 | 值  | 备注     |
| ------ | ---- | --- | -------- |
| number | 整数 |     | 奖励的id |

### 返回

| 参数 | 类型   | 值                  | 备注           |
| ---- | ------ | ------------------- | -------------- |
| code | 整数   | `0`(失败),`1`(成功) | 请求的状态     |
| msg  | 字符串 |                     | 状态的具体说明 |

```json
{
    "code": 0,
    "msg": "你还没有达到领取条件！"
}
```
