## 评论点赞


### 地址
```
/module/action/like-post.php
```
### 要求
- 用户已登录
- 非黑名单用户

### 方法
- POST

### 参数

| 参数       | 类型 | 值  | 备注     |
| ---------- | ---- | --- | -------- |
| comment_id | 整数 |     | 评论的id |
| type       | 整数 | 2   | 必须为2  |


### 返回
| 参数 | 类型   | 值    | 备注 |
| ---- | ------ | ----- | ---- |
| code | 整数   | `0`(失败),`1`(成功) | 请求的状态 |
| msg  | 字符串 |       |   描述 |

```json
{
    "code":"1",
    "msg": "点赞成功！"
}
```
